## Setup

### Begin

1. Follow the initial setup steps in the [root README.md file](/README.md#setup)
2. Change directories to `frontend`

   ```shell
   $ cd frontend
   ```

3. Install Dependencies

   ```shell
   $ npm install
   ```

### Start local development

```shell
$ npm start
```
